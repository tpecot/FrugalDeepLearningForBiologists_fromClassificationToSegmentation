# FrugalDeepLearningForBiologists_fromClassificationToSegmentation
3-day workshop to teach biologists how to train and process deep convolutional neural networks for image classification and segmentation to be run on laptops (even without GPU, although it'll take more time)

Deep learning has demonstrated astonishing classification and segmentation results in microscopy, outperforming all existing approaches. While many codes are publicly available, they require expertise that most biologists lack. The goal of this workshop is to learn how to train and process frugal deep convolutional neural networks for image classification and image segmentation. More specifically, participants will learn how to install python packages and run Marimo notebooks, train deep learning classifiers to classify images with transfer learning, annotate images in QuPAth and export them in a suited format, train U-Net models and convert them to pytorch scripts so they can be used directly in QuPath.<br>

This is a 3-day workshop. Slides describing the background required to understand the methods and links to video tutorials are available in Courses.
